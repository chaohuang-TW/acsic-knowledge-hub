# 官方來源方法論

## 來源優先序

1. 機構官方法規、制度文件及年度報告。
2. 機構官方介紹、業務及治理網頁。
3. 政府主管機關官方資料。
4. ACSIC 官方會員索引，用於會員與 Observer 身分核對。

搜尋引擎只用於定位官方頁面，搜尋摘要不進入資料集。

## 欄位處理

- 每個來源保存標題、文件類型、發布者、網址、文件日期、年份、章節及查閱日期。
- 官方頁面未明示時使用 `null` 或「官方資料未揭露」。
- 保證成數、上限、費率及營運量等動態數字，只有同時具備資料日期、來源及最後查證日期時才可收錄。
- 不把政策融資、信用保險、聯合組織與直接信用保證承作機構視為相同功能。

## 查證狀態

- `verified`：核心機構定位、法源或設立資訊及 ACSIC 身分有官方一手來源。
- `partially_verified`：機構與 ACSIC 身分已確認，但治理、資金或方案條件仍有缺口。
- `pending_verification`：只有初步官方入口，尚未完成核心欄位查核。

正式引用及決策仍應回到原始來源，確認頁面或文件是否更新。

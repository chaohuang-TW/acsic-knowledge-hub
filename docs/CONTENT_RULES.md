# 公開內容規則

## 可公開內容

- 完全虛構且明確標示 DEMO 的國家、機構、制度、數字與來源。
- 介面、資料模型、查證狀態、可信度與報告範本的示範。
- 一般性的資料治理方法及公開審查流程。

## 禁止公開內容

- 私人核心 repository 的任何檔案、片段、路徑、歷史、資料或同步機制。
- 真實保證戶、個人資料、內部人名、職稱關係、聯絡資訊與私人電子郵件。
- 未公開公文、會議、簡報、底稿、政策判斷、財務或業務數據。
- Token、API Key、密碼、Cookie、私鑰、本機路徑與環境設定。
- 真實機構名稱搭配虛構制度數字。

## DEMO 資料規範

- 資料物件包含 `demo: true`。
- 畫面、比較、報告及下載檔均顯示 DEMO 與非官方聲明。
- 使用虛構國家及虛構機構名稱。
- 示範來源使用 `.invalid` 網域，並明確寫明完全虛構。
- 不使用精確虛構 KPI、金額或保證比例裝飾畫面。

## 來源規範

- 分開示範一手來源及示範二手來源。
- 記錄標題、發布者、網址、存取日期與說明。
- 缺少來源時標記待查證，不得推測補入。

## 非官方定位

所有頁面保留完整聲明：本網站是個人 AI 知識管理與介面設計實驗，不是任何政府機關、基金會或國際組織的官方網站，內容不構成政策、金融或法律建議。

## 視覺邊界

- 禁止官方 Logo、徽章、印章、旗幟與政府識別色組合。
- 禁止模仿政府入口網站、金融機構或國際組織的資訊架構與品牌。
- 只使用專案自行生成且不含真實機構元素的抽象視覺。
